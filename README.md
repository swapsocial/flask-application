# Flask App

This is a simple Flask application that prints "Hello World" when accessed on the homepage.

## How to Run

1. Install dependencies:

    ```bash
    pip install flask
    ```

2. Run the app:

    ```bash
    python app.py
    ```

3. Open your browser and visit:

    ```
    http://127.0.0.1:5000/
    ```

You should see "Hello World with version".
